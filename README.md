# E-commerce-Customer-support-chatbot-AI-And-ML
The ElectroMart system is divided into three major modules to ensure structured development and easy maintenance.
📦 ElectroMart – Smart E-commerce Web App
🚀 Project Overview
ElectroMart is a modern e-commerce web application developed to provide a smooth and interactive shopping experience. Users can browse products, search items, add to cart, and interact with a smart product assistant.
✨ Features
🛍️ Product browsing and search functionality
📂 Category-based filtering
🛒 Shopping cart (add, remove, update items)
🔐 User & Admin login system
🤖 Smart Product Assistant (with voice support)
📱 Responsive UI design
